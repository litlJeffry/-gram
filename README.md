# 🌮gram - Полноценный мессенджер

Реальный мессенджер с регистрацией, уникальными ID и общением между пользователями!

## 🚀 Возможности

- ✅ **Регистрация** с присвоением уникального ID (1, 2, 3, 4...)
- ✅ **Авторизация** по логину и паролю
- ✅ **Обмен сообщениями** в реальном времени
- ✅ **Отправка изображений** (jpg, png, gif, webp)
- ✅ **Статусы онлайн/офлайн**
- ✅ **Индикатор печати**
- ✅ **Прочитано/доставлено**
- ✅ **Сохранение истории** чатов
- ✅ **Поиск пользователей**

## 📦 Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/your-username/tacogram.git
cd tacogram

# Установите зависимости
npm install

# Запустите сервер
npm start

# Откройте в браузере
# http://localhost:3000
